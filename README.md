# wunderland-scraper
### To Use
#### Specific Date(s)
` ruby scraper.rb [ date ] [ date ( optional ) ] [ date ( optional ) ] `

e.g 

` ruby scraper.rb 31/08/2014 `

` ruby scraper.rb 31/08/2014 21/04/2013 `

#### Date Range
` ruby scraper-range.rb [ start-date ] [ end-date ] `

e.g

` ruby scraper-range.rb 23.07.2010 22.09.2014 `
